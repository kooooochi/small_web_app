# components/

UIコンポーネント

## 概要

再利用可能なUIコンポーネントを格納するディレクトリです。

## コンポーネント一覧

### Button/
汎用ボタンコンポーネント
- `index.js` - ボタンロジック
- `styles.css` - スタイル定義

### Header/
ページヘッダーコンポーネント
- `index.js` - ヘッダーロジック
- `styles.css` - スタイル定義

## 使用方法

```javascript
import Button from './components/Button';
import Header from './components/Header';
```
