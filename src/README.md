# src/

アプリケーションのソースコード

## 概要

このディレクトリには、アプリケーションのメインコードが含まれています。

## ディレクトリ構成

- **[components/](./components/README.md)** - 再利用可能なUIコンポーネント
- **[utils/](./utils/README.md)** - ヘルパー関数とユーティリティ
- **[services/](./services/README.md)** - API通信やビジネスロジック

## 開発ガイドライン

- コンポーネントは機能ごとに分離
- 共通処理はutilsに配置
- 外部API通信はservicesに集約
